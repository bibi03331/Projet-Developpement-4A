*   Serveur Node.js   *

app.js : Contient le code du serveur

index.html : Page de test permettant de faire des tests unitaires sur les moteurs

package.json : Spécification de l’appliation Node.js
start.sh : Script permettant de lancer le serveur Node.js ainsi que le serveur de streaming en tâche de fond
